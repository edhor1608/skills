# Project Bootstrap Brand Pack

Create a coherent project starter package from a short brief.

## Deliverables

- Project name options
- Selected name rationale
- Phonetic rendering
- License recommendation with tradeoff summary
- GitHub description
- Starter README.md

## Workflow

1. Extract project purpose, audience, and constraints.
2. Produce 5 to 10 strong name candidates.
3. Add phonetic rendering for top candidates.
4. Recommend one license based on business intent.
5. Draft GitHub description.
6. Draft concise README with setup and scope.

## Guardrails

- Avoid generic AI-style names.
- Keep README practical and scannable.
- Explain license choice in one short paragraph.
